### 2. 어셈블리 명령어

#### - 데이터 전송 명령어

한 장소에서 다른 장소로 데이터 전송하는 명령어

입출력 명령어 포함

| 이름       | 니모닉 | 기능                                                  |
| ---------- | ------ | ----------------------------------------------------- |
| $Load$     | $LD$   | 기억장치로부터 레지스터로의 전송                      |
| $Store$    | $ST$   | 레지스터로부터 기억장치로의 전송                      |
| $Move$     | $MOVE$ | 레지스터로부터 다른 레지스터로의 전송                 |
| $Exchange$ | $XCH$  | 두 레지스터 간 / 레지스터와 기억장치 간의 데이터 교환 |
| $Push$     | $PUSH$ | 기억장치의 스택과 레지스터 간의 데이터 전송           |
| $Pop$      | $POP$  | -                                                     |
| $Input$    | $IN$   | 레지스터와 입출력장치 간의 데이터 전송                |
| $Output$   | $OUT$  | -                                                     |

#### - 데이터 처리 명령어

##### - 산술연산 명령어

사칙연산에 대한 산술 명령어

| 산술 명령어                | 니모닉 | 기능              |
| -------------------------- | ------ | ----------------- |
| $Increment$                | $INC$  | 1증가             |
| $Decrement$                | $DEC$  | 1감소             |
| $Add$                      | $ADD$  | $+$               |
| $Subtract$                 | $SUB$  | $-$               |
| $Multiply$                 | $MUL$  | $\times$          |
| $Divide$                   | $DIV$  | $\div$            |
| $Add$ $with$ $carry$       | $ADDC$ | 캐리를 포함한 $+$ |
| $Subtrack$ $with$ $borrow$ | $SUBB$ | 빌림을 포함한 $-$ |
| $Negate$                   | $NEG$  | 2의 보수          |

##### - 논리연산 명령어 & 비트 처리 명령어

2진 연산 수행

2진 부호화 정보를 포현하는 비트 그룹이나 개별 비트를 처리하는 데 사용

비트값 0으로 만들거나 / 기억장치 레지스터에 저장된 오퍼랜드에 새로운 비트값을 삽입하는 것 등 가능

| 논리 명령어          | 니모닉 | 기능                 |
| -------------------- | ------ | -------------------- |
| $Clear$              | $CLR$  | 모두 비트 0으로 리셋 |
| $Set$                | $SET$  | 모두 비트 1으로 리셋 |
| $Complement$         | $COM$  | 모두 비트 반전       |
| $ADD$                | $AND$  | 비트별 AND 연산      |
| $OR$                 | $OR$   | 비트별 OR 연산       |
| $Exclusive-OR$       | $XOR$  | 비트별 XOR 연산      |
| $Clear$ $carry$      | $CLRC$ | 캐리비트의 리셋      |
| $Set$ $carry$        | $SETC$ | 캐리비트의 세트      |
| $Complement$ $Carry$ | $COMC$ | (반전)보수           |

##### - 시프트 명령어

| 종류          | 시프트 명령어                   | 니모닉 | 기능                                              |
| ------------- | ------------------------------- | ------ | ------------------------------------------------- |
| 논리적 시프트 | $Logical$ $shift$ $right$       | $SHR$  | 오른쪽 시프트(왼쪽 남는 비트 0으로 채움)          |
|               | $Logical$ $shift$ $left$        | $SHL$  | 왼쪽 시프트(오른쪽 남는 비트 0으로 채움)          |
| 산술적 시프트 | $Arithmetic$ $shift$ $right$    | $SHRA$ | 부호비트 고정(왼쪽 남는 비트 부호비트로 채움)     |
|               | $Arithmetic$ $shift$ $left$     | $SHLA$ | 부호비트는 고정(오른쪽 남는 비트 0으로 채움)      |
| 회전형 시프트 | $Rotate$ $right$                | $ROR$  | 오른쪽으로 순환(버려지는 비트는 다시 왼쪽 비트로) |
|               | $Rotate$ $left$                 | $ROL$  | 왼쪽으로 순환(버려지는 비트는 다시 오른쪽 비트로) |
|               | $Rotate$ $right$ $with$ $carry$ | $RORC$ | 캐리 포함 오른쪽 순환                             |
|               | $Rotate$ $left$ $with$ $carry$  | $ROLC$ | 캐리 포함 왼쪽 순환                               |

##### - 프로그램 제어 명령어

프로그램 수행 흐름 제어

다른 프로그램 세그먼트(segment)로 분기할 수 있는 능력 제공

| 제어 명령어                    | 니모닉 | 기능                                               |
| ------------------------------ | ------ | -------------------------------------------------- |
| $Branch$                       | $BR$   | 조건 혹은 무조건적으로 유효주소로 분기             |
| $Jump$                         | $JMP$  | -                                                  |
| $Skip$ $next$ $instruction$    | $SKP$  | 조건이 만족되면 다음 명령어를 수행하지 않고 넘어감 |
| $Call$ $procedure$             | $CALL$ | 서브루틴 호출                                      |
| $Return$ $from$ $procedure$    | $RET$  | 서브루틴 실행 후 복귀                              |
| $Compare$ $(by$ $subtraction)$ | $CMP$  | 두 오퍼랜드의 뺄셈을 통해 상태 레지스터의 값 변환  |
| $Test$ $(by$ $ANDing)$         | $TEST$ | 논리 AND 연산만 구현                               |
